MyReads is a React application developed as the final assessment project for Udacity's **React Fundamentals** course.

The application allows users to organise their personal library by placing books into one of three shelves:

- **Currently Reading**
- **Want to Read**
- **Read**

Users can move books between shelves by clicking the green shelf selector on each book and choosing a new shelf.

New books can be added by clicking the **Add a Book** button in the bottom-right corner of the application. The search page allows users to search for books by title or author and add them directly to one of the three shelves. Any changes made on the search page are immediately reflected on the main bookshelf page.

## Installation and Setup

1. Clone the repository:

   ```bash
   git clone https://github.com/Lee-Powell/myreads-react.git
   ```

2. Navigate to the project directory:

   ```bash
   cd starter
   ```

3. Install the project dependencies:

   ```bash
   npm install
   ```

4. Start the development server:

   ```bash
   npm start
   ```

5. Open your browser and visit:

   ```
   http://localhost:3000
   ```

## Technologies Used

- React
- React Router
- Udacity Books API
